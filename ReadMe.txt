1. Open the SnowGL Solution file in VS2019 (v141 toolset required)
2. Ensure x64 and Release are selected for build
3. Build the project
4. Copy the SDL2.dll file from SDKs/Lib64/ to the executable directory
5. Run from Visual Studio

Alternatively, to run from the executable:
1. Copy the resources folder from SnowGL/ to the executable folder
2. Run the executable
